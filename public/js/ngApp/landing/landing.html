Welcome to oSource...

Helping developers find open-source projects with ease


Searching for open-source projects can be a daunting task...


With oSource our goal is to simplify this process. Using oSource allows every user to search GitHub's repos for open-source projects that they may want to contribute to. Utilizing GitHub to list your own repos allows the user to add a repo that they may want help with to the oSource's searchable database. Once added to our database, a user can add information to the repo allowing others to find it by keywords, skill level, and languages.


Here is how to get the most out of oSource

  • Login using GitHub if you want to get the most out of oSource
    (*You should link LinkedIn and/or Bitbucket from the settings page*)
  • Navigate to the Dashboard to see your repos then add them to the searchable Osource database
  • Search GitHub for projects that you may want contribute to!
  • Reach out the repo owner for certain tasks or code away and submit a pull request
  • Search oSource's database for projects that have been listed, tasks that need to be completed should be listed
